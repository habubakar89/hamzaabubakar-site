# Hamza Abubakar - Portfolio

A pixel-perfect portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- Responsive two-column layout (sticky sidebar on desktop)
- Smooth scrolling navigation with scrollspy
- Cursor spotlight effect
- Full accessibility support
- SEO optimized

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Inter font via next/font

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Editing Content

All content is in `src/data/site.ts`. Edit this file to update:
- Hero section (name, role, tagline)
- About paragraphs
- Experience entries
- Projects
- Writing/articles
- Social links

## Deploy to Vercel

```bash
vercel --prod
```
