evaluation frameworks that track proxy metrics instead of user outcomes —

LLM is often the easiest part of the system to swap. The surrounding
infrastructure — streaming pipelines, observability, retrieval architecture,
structured output enforcement — is what determines whether users can
actually rely on the system."

proud of are all in active use by teams I've never worked with directly,
built on patterns I extracted from my own production experience."
